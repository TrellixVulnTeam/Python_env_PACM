
# Python Virtual Env - Notes to Self



## Creation:  (Windows)
### -> py -m venv Python_env
#### - Or Just Clone this Repo and Activate

## Activation: (Windows)
### *** Run VScode as Admin

### -> Set-ExecutionPolicy RemoteSigned
### -> ..\Python_env\Scripts\activate

#### - Or Just Clone this Repo and Activate

## Activation: (Linux)
### -> sudo source env/bin/activate
